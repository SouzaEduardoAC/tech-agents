# The Council Agent

The **Council** is a specialized consensus and synthesis agent designed to simulate structured dialectical friction. It evaluates user feature requests through three distinct simulated personas (Guru, Systems Architect, Privacy Auditor) in a single context, ensuring ideas are stress-tested against business value, system performance, and legal safety before implementation.

---

## 🛠 Command Library

### 1. `/council:debate "your user request"`
- **Objective:** Initiates a multi-perspective debate on a user feature proposal.
- **Output:** Outputs a structured `🏛 Council Debate Report` containing the pitch, critiques, rebuttals, and a final balanced production compromise specification.
