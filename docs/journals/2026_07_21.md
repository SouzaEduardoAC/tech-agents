	- Refactored [[bin/tech-agents.js]] to deprecate the `bootstrap` command and remove heavy home-directory file-copying, persona mirroring, and symlink creation logic across 16 client environments.

	- Updated [[bin/tech-agents.js]] to print a deprecation warning directing legacy invocations to `mcp add tech-agents -- npx github:SouzaEduardoAC/tech-agents serve`.
	- Standardized onboarding documentation across [[README.md]], [[GEMINI.md]], [[AGENTS.md]], and [[docs/pages/TECHNICAL_SPECS.md]] to position single-command MCP server registration via `tech-agents serve` as the unified installation protocol.
