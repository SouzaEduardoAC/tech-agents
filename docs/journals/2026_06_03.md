- # MCP Command Aliasing and Agent Commands Discovery
	- **Implement command aliases**: Implemented command alias resolving in `index.js`'s `call_agent_command` handler to transparently route commonly guessed/inaccurate command names (e.g. `squad:create` to `squad:run`, `architect:discovery` to `architect:create`, `decoder:synthesize` to `decoder:export`) to the actual TOML definition files, covering all 13 specialized agents. (ref: `index.js`)
	- **List agents and their commands**: Enhanced the `list_agents` tool in `index.js` to scan command directories, exclude the shared `common` folder, and return a clean listing of all active agents alongside their available commands. (ref: `index.js`)
	- **Self-correcting error messages**: Enhanced `call_agent_command` error throwing to output a list of valid commands for the target agent upon encountering an invalid command name. (ref: `index.js`)
	- **Refine Schema and Parameter Descriptions**: Updated tool definitions in `ListToolsRequestSchema` to guide agentic clients (like Claude Code) toward correct command query workflows and clean command expectations. (ref: `index.js`)
