- # Feature: Serve MCP Usage Guide via `list_agents`
	- **Problem**: Previously, `list_agents` only returned a directory listing of agents and their commands. The complete MCP Usage Guide and Decision Flowchart (`common/skills/mcp_usage_guide.md`) was only injected as a skill in agent prompt assembly, making it hard to retrieve directly or read to understand client-server usage prior to execution.
	- **MCP Integration**: Modified `index.js` to serve `common/skills/mcp_usage_guide.md` directly via the `list_agents` tool when the file is present. Keep directory scanning of agents and commands as a fallback.
	- **Description Update**: Updated `list_agents` tool description in the tool registration block to inform clients they can use this tool to retrieve the complete MCP Usage Guide & Decision Flowchart.
	- **Test Parity**: Updated the integration test suite in `test/mcp-integration.mjs` to dynamically parse the 13 agents from the new markdown-formatted guide response or fall back to the listing format.
	- **Documentation Update**: Added entry to `CHANGELOG.md` and updated `docs/pages/registry.md`. (ref: `index.js`, `test/mcp-integration.mjs`, `CHANGELOG.md`, `docs/pages/registry.md`)
