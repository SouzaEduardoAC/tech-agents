- # Integrated MCP Sampling & Managed Loop Execution
	- Worked on adding dynamic MCP Sampling support to the `agent-hub` MCP server, allowing client-driven conversation loops via `sampling/createMessage` for SSO/token-based enterprise environments (like AntiGravity or Codex) where local API keys are unavailable.
	- Registered the `run_agent_loop` tool in `index.js` and implemented the execution handler.
	- Built a server-side XML action parser supporting local filesystem reads (`<read_file>`), writes (`<write_file>`), and shell commands (`<run_command>`) with strict path containment checks against `projectRoot` directory traversal.
	- Implemented prompt pinning in the managed loop to feed the full agent persona and standards context on every turn, completely mitigating Codex instruction/prompt drift.
	- Designed and successfully ran integration tests in `test/mcp-sampling-test.js` to assert both fallback errors (when sampling is unsupported) and complete multi-turn local tool execution loops (when sampling is supported).
	- Updated `TECHNICAL_SPECS.md`, `registry.md`, `README.md`, `AGENTS.md`, and `GEMINI.md` to document the new tool and usage protocols.
